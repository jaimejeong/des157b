Capitalize First Letter

This function is used for capitalizing the first letter of each word in a string. For my Library Liberty project, I wanted to capitalize the first letter of user's name, and later I got intrigued on how to capitalize each word when there's a space between. I've used this website to understand and write the code.

Named Function

            function firstUpperCase(str) {
                str = str.toLowerCase();
                const arr = str.split(" ");
    
                for (let i = 0; i < arr.length; i ++) {
                    arr[i] = arr[i].charAt(0).toUpperCase() + arr[i].slice(1);
                }
                return arr.join(" ");
            }
